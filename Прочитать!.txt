Файл для запуска FTPclient.py

Остальные файлы - подключаемые модули, сами по себе они работать не будут.
Рабочая дирректория на компьютере папка files (если по какой-то причине она будет удалена, она создатся автоматически при запуске)
папка image содержит картинки и звуковой файл, используемые программой, по желанию их можно поменять, но главное сохранить названия.
папка setting содержит файлы с настройками - данные ftp сервера, сохраненный путь. (они могут быть удалены, программа их создает автоматически и при изменении настроек она их меняет)
для работы программы в виде питоновских файлов, на интерпритаторе питона должна быть библиотека PyQt5 или PyQt6
